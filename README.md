## Instalation :
```bash
> npm i abir-downloader
```

## Example (tikdown)
```js
const tikdown = require('abir-downloader');

const videoUrl = 'url'; //past your video link

tikdown(videoUrl, (error, result) => {
    if (error) {
        console.error('Error:', error);
    } else {
        console.log(result);
    }
});

```
